# Design UI UX

Folder ini berisi desain antarmuka (UI/UX) aplikasi Smart Billiard Lighting.

## Isi Folder

- File Figma
- Mockup Desktop
- Asset Icon
- Screenshot Progress

## Halaman

### Dashboard
Menampilkan informasi utama sistem:
- Status lampu
- Mode sistem
- Nomor meja aktif
- Countdown waktu
- Status koneksi Arduino
- Log sistem

### Billing
Digunakan untuk pengelolaan penyewaan meja:
- Pemilihan meja
- Pemilihan durasi bermain
- Perhitungan tarif
- Mulai billing
- Tambah waktu
- Selesai billing
- Monitoring billing aktif
