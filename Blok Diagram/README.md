Technical Solution
1. Integrasi Sistem
 Aplikasi kasir terhubung dengan ATmega2560 melalui UART Serial.
 Status meja dikendalikan dari dashboard kasir.
2. Otomatisasi Proses
 Timer dan perhitungan biaya berjalan secara real-time.
 Lampu meja mengikuti status permainan secara otomatis.
3. Kontrol Aktuator
 Relay 5V digunakan untuk mengendalikan lampu meja biliar.
 Mendukung mode otomatis dan manual.
